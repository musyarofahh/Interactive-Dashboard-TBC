# Interactive-Dashboard-TBC
This project demonstrates how to build an interactive dashboard using Power BI to analyze and visualize data dynamically. The dashboard is designed to help users explore datasets easily, identify trends, and make data-driven decisions.

## 🎯 Features
- Interactive visualizations (bar charts, line charts, pie charts, maps).  
- Dynamic filtering with **Slicers** and **Drill-through**.  
- Automated summaries using **DAX measures**.  
- Clean and user-friendly layout.  
- Applicable for sales reports, financial analysis, or operational monitoring.  

## 🛠️ Tools & Technologies
- **Power BI Desktop**  
- **DAX (Data Analysis Expressions)**  
- **Power Query** for data cleaning and transformation  
- **Excel/CSV** as data sources  

## 🚀 How to Use
1. Open the `.pbix` file in Power BI Desktop.  
2. Navigate to the **Dashboard** page.  
3. Use slicers or filters to interact with the data.  
4. Explore charts and KPIs to gain insights.  

## 📂 Project Structure
- `Data/` → Raw dataset (Excel/CSV).  
- `Dashboard.pbix` → Main Power BI file.  
- `README.md` → Project documentation.  

## 📈 Example Use Cases
- Monthly sales performance analysis.  
- Financial reporting and forecasting.  
- Employee productivity monitoring.  
- Research data visualization.
